# vuln3r
Easy Vulnerability Detection &amp; Hints With CVE Number For Exploitation

### Built With

* [Python3]

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites 
* python3-pip -y
  ```
  apt install python3-pip
  ```
* nmap
  ```
  apt install nmap -y
  ```
* git 
  ```
  apt install git -y
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Ba-hub/vuln3r.git
   ```
2. Give Executable Permission 

* For Kali Linux,Ubantu, Debian

   ```
   python3 vuln3r.py -h
   ```
*  Example
   ```
   python3 vuln3r.py -u https://google.com -o google.txt
   ```
